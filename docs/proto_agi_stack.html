Stack
Perception
Text, code, files, structured data, later multimodal inputs.
World Model
Projects, tools, environment state, branches, facts, and causal relationships.
ARC Memory + Receipts
Episodic, semantic, procedural, and strategic memory with provenance.
Goal Engine
Goal creation, prioritization, decomposition, and tracking.
Planner
Turns goals into executable action graphs and revises plans after failure.
Action / Tool Layer
Shell commands, scripts, build pipelines, file ops, APIs, wrapped tools.
Evaluation
Scores actions, detects waste/failure, and records what should be remembered.
Learning
Promotes successful patterns into reusable procedures and better strategies.
Persistent Agent Loop
The always-running process that ties the whole system together across time.
